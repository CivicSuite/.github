<h1 align="center">Municipal software cities can actually own.</h1>

<p align="center">
  CivicSuite is an open-source, local-first municipal operations suite. One Windows installer brings records, meetings, municipal code, public notices, accessibility workflows, and the shared platform onto the city's own hardware—without a vendor cloud, telemetry, or per-seat pricing.
</p>

### The city-core suite today

| Module | Municipal work | Current role |
|---|---|---|
| [CivicCore](https://github.com/CivicSuite/civiccore) | Identity, audit, retention, local AI, search, and shared services | Shared platform |
| [CivicRecords AI](https://github.com/CivicSuite/civicsunshine) | Public-records intake, search, and response drafting | City-core |
| [CivicClerk](https://github.com/CivicSuite/civicmeetings) | Meetings, agendas, packets, minutes, and votes | City-core |
| [CivicCode](https://github.com/CivicSuite/civiccode) | Searchable municipal code and ordinance access | City-core |
| [CivicNotice](https://github.com/CivicSuite/civicnotice) | Public-notice creation and publishing support | City-core |
| [CivicAccess](https://github.com/CivicSuite/civicaccess) | Plain language, translation drafts, and accessibility review | City-core |

> **Honest status:** CivicSuite Windows Local v1.0.2 is a GA candidate open for public beta. It is ready for hands-on evaluation, but it is **not production-, city-, or procurement-ready**. The broader module catalog includes queued, foundation, and planned work; repository count is not product maturity. [Read the module-by-module status.](https://github.com/CivicSuite/civicsuite/blob/main/STATUS.md)

### Why local-first

- **City-held data.** Documents, audit history, the database, and the AI model stay on the municipality's hardware.
- **Clerk-first workflows.** The active suite starts with records, meetings, code, notices, and accessible public information.
- **Human authority.** AI can draft and assist; humans decide. CivicSuite does not auto-release, auto-deny, auto-redact, or make legal determinations.
- **Open foundations.** Code is Apache 2.0; documentation is CC BY 4.0. Cities retain the right to inspect, modify, operate, and fork the software.
