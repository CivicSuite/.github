# CivicSuite project status

**Profile snapshot:** July 2026. The authoritative status is maintained in the [`civicsuite` umbrella repository](https://github.com/CivicSuite/civicsuite/blob/main/STATUS.md).

## Current public statement

CivicSuite Windows Local v1.0.2 is a **GA candidate open for public beta**. The six-module city-core build has passed the project's clean-machine acceptance work and is available for real hands-on evaluation.

It is **not production-ready, city-ready, procurement-ready, or a full-suite release**. CivicSuite does not currently document a live municipal deployment.

## Maturity groups

| Group | Meaning |
|---|---|
| Shared platform | CivicCore, required by every module and always installed. |
| Released city-core | Records, meetings, municipal code, public notices, and accessibility workflows included in the Windows Local beta. |
| Queued Tier 2 | Early foundations with corrected version labels; deeper product work is queued. |
| Early scaffold / foundation | Schema, package, sample workflow, or test foundations; not city-ready products. |
| Planned | Specification exists, but no runtime repository exists yet. |

Repository count, release tags, and version numbers do not by themselves establish product maturity. Use the public status matrix and release evidence.
