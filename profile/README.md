# CivicSuite

**Open-source municipal software that runs on the city's own hardware.** No cloud, no telemetry, no per-seat pricing — the city owns its data, its documents, and even its AI model.

### 🏛️ Start here → **[civicsuite.github.io/civicsuite](https://civicsuite.github.io/civicsuite/)**

**Current release:** [CivicSuite Windows Local v1.0.2](https://github.com/CivicSuite/civicsuite/releases/latest) — one Windows installer sets up the whole six-module city-core suite: shared platform, public records (FOIA), meetings & minutes, municipal code, public notices, and accessibility — with a bundled local database and a local AI model. A clerk installs it with a double-click; nothing requires a terminal, Docker, or a cloud account.

**Honest status:** a GA-candidate **public beta**. Validated end to end on clean Windows machines; the one remaining gate to general availability is code signing (in progress). It is not yet production-, city-, or procurement-ready — see [STATUS.md](https://github.com/CivicSuite/civicsuite/blob/main/STATUS.md) for the module-by-module truth.
